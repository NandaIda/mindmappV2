Help & Instructions

Welcome to MindMapp!

MindMapp is an intuitive mind mapping application that helps you organize your thoughts, brainstorm ideas, and create visual representations of your thoughts. With drag-and-drop functionality, keyboard shortcuts, and seamless import/export capabilities, MindMapp makes it easy to capture and develop your ideas.

Getting Started
  • Create nodes: Click the "Add" button in the bottom navbar or use the + key
  • Edit nodes: Double-click on any node to edit its text
  • Move nodes: Click and drag any node to reposition it
  • Delete nodes: Select a node and click "Del" button or press -
  • Navigate: Use mouse wheel to zoom, click and drag on empty space to pan
Keyboard Shortcuts
Action Shortcut
Add child node + or =
Delete selected node - or _
Undo Ctrl+Z (Windows) / Cmd+Z (Mac)
Redo Ctrl+Y (Windows) / Cmd+Shift+Z (Mac)
Save mindmap Ctrl+S (Windows) / Cmd+S (Mac)
Load mindmap Ctrl+O (Windows) / Cmd+O (Mac)
Features
  • Drag & Drop: Easily move nodes around to organize your mind map
  • Touch Support: Full support for mobile devices and touchscreens
  • Import/Export: Save and load your mind maps as .mind files
  • Undo/Redo: Full history support with up to 50 actions
  • Zoom & Pan: Navigate large mind maps with ease
